# Image File Names Guide

## Clinic Interior Image

**File Name:** `clinic.jpg`  
**Location:** `/public/brand/clinic.jpg`

This image is used in:
- Home page hero section
- About page
- Gallery page
- Virtual tour section

**Recommended Specifications:**
- Format: JPG or PNG
- Size: 1920x1080px or larger
- File size: Under 2MB (optimize for web)

## Logo Image

**File Name:** `happy-optics-logo.png`  
**Location:** `/public/brand/happy-optics-logo.png`

**Recommended Specifications:**
- Format: PNG (with transparent background)
- Size: 512x512px or larger
- File size: Under 500KB

## Gallery Images

**Location:** `/public/gallery/`

Recommended file names:
- `clinic-interior-1.jpg`
- `clinic-interior-2.jpg`
- `eyewear-showcase-1.jpg`
- `eyewear-showcase-2.jpg`
- `patient-care-1.jpg`
- `team-photo.jpg`

**Note:** If images are missing, the gallery will show placeholder SVGs automatically.
