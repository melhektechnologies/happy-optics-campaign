## File Locations

```
public/
  brand/
    ├── clinic.jpg          ← Replace with new clinic interior photo
    ├── team.jpg            ← Add new team photo here
    ├── happy-optics-logo.png
    └── README.md
```

## Image Specifications

### Recommended Sizes:
- **clinic.jpg**: 1920x1080px or larger (16:9 or 4:3 aspect ratio)
- **team.jpg**: 1920x1080px or larger (16:9 or 4:3 aspect ratio)

### File Format:
- JPG or PNG
- Optimized for web (under 2MB recommended)

## After Adding Images

Once you've added the images:
1. The home page will automatically show the new clinic photo
2. The About page will show both the clinic photo and the new team photo
3. No code changes needed - just replace/add the image files!
